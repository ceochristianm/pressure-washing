# Monthly Reviews

Claude writes dated analysis files here (YYYY-MM.md) after each month's data is in.
Weekly mini-reviews can live here too (YYYY-MM-DD-weekly.md).

Every review answers the 6 standing questions from CLAUDE.md:
1. Cash vs. Amex clock  2. Revenue vs. roadmap target  3. CPL/CPJ by channel
4. Close rate + speed-to-lead  5. AJS vs. $550 target  6. Biggest leak + #1 next action

Verdicts must be direct: what worked, what didn't, what we were wrong about.
