# Offline / Guerrilla Marketing

## Yard signs (~$100 — the one paid guerrilla item)
- ~20 corrugated 18x24 signs. White background, bold red/blue text ONLY:
  "HOUSE WASHING" + phone number. No logo/site — readable at 45 mph.
- Deploy Thu/Fri evening: stoplights + medians at gated community entrances in
  target areas. Expect some removal by code enforcement — replaceable consumable.
- Track: ask every caller "how'd you hear about us" → data/leads.csv source column.

## Door hangers / flyers
- Around every active job: 5 nearest homes (2+2 sides, 3 across, more if momentum)
- Angle: "We're washing your neighbor's house at [#] today — same-street discount
  if booked this week." Specificity beats generic flyers.
- Daily baseline when not on a job: 20 hangers in target neighborhoods.

## The 10-door knock (gated communities)
- Timing: BEFORE starting the job (fresh, not sweaty; morning knocks OK — retirees
  and WFH answer). The running rig later is the credibility proof for anyone who watched.
- Script in sales-scripts.md.

## Commercial walk-ins (cash-flow filler, 3/day baseline)
- Strip malls/restaurants/gas stations with filthy entrances
- Pitch = slip-and-fall liability, not aesthetics (script in sales-scripts.md)
- NOTE: commercial requires the CCC insurance endorsement + is sales-taxable —
  fine for small storefront jobs, defer big contracts to Month 3+.
